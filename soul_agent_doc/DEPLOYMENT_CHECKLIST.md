## 部署前准备

### 环境与基础设施
- [ ] 服务器已购买并配置（推荐：4核4G，Ubuntu 22.04）
- [ ] 域名已备案并解析到服务器IP
- [ ] SSL证书已申请（推荐Let's Encrypt免费证书）
- [ ] 防火墙已配置（只开放22、80、443端口）
- [ ] SSH密钥登录已配置（禁用密码登录）

### API Key准备
- [ ] 智谱AI API Key已申请并测试可用
- [ ] DeepSeek API Key已申请（备用）
- [ ] 高德地图JS SDK Key已申请
- [ ] 高德地图Key已配置使用限制（域名/IP）
- [ ] 所有API Key已记录到安全位置

## 代码部署

### 1. 代码拉取
- [ ] `git clone` 项目到服务器 `/var/www/zheilitrip/app`
- [ ] 切换到正确的分支（main/stable）
- [ ] 确认 `.gitignore` 已包含 `.env`

### 2. 环境变量配置
- [ ] 复制 `.env.example` 为 `.env`
- [ ] 填写生产环境API Key
- [ ] 设置 `APP_ENV=production`
- [ ] 设置 `.env` 文件权限为 `600`（仅root可读）
- [ ] 验证Key格式正确（无多余空格/换行）

### 3. 后端部署
- [ ] Python虚拟环境已创建（`.venv`）
- [ ] `pip install -r requirements.txt` 成功
- [ ] `gunicorn` 已安装
- [ ] systemd服务文件已创建
- [ ] 服务已启动并设置开机自启
- [ ] 健康检查通过：`curl http://localhost:8000/health`
- [ ] API文档可访问：`http://localhost:8000/docs`
- [ ] rich-plan接口测试成功

### 4. 前端部署
- [ ] `npm install` 成功
- [ ] `npm run build` 成功
- [ ] `h5/dist/` 目录已生成
- [ ] 高德地图Key在 `dist/index.html` 中正确显示
- [ ] 静态资源路径正确

### 5. Nginx配置
- [ ] Nginx配置文件已创建
- [ ] 域名已替换为实际域名
- [ ] SSL证书路径正确
- [ ] 后端API代理配置正确
- [ ] SPA路由支持已配置（try_files）
- [ ] Gzip压缩已启用
- [ ] 静态资源缓存策略已配置
- [ ] `nginx -t` 测试通过
- [ ] Nginx已重启
